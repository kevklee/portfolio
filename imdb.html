IMDb

Top movie list and forums redesign

About

The current IMBd top movies list is not as organized and user friendly as it could be. The design displayed here is to make the top movies list page more intuitive and easy for users.

Problem

The IMDb top movie list in its various forms currently has many problems. One of which is complex navigation that gets users lost. Also, a lot of the user interface is cluttered and a several buttons have no purpose. The biggest challenge is that the users cannot seamlessly navigate the user interface.

Solution

The solution was based on the "don't make me think" principle. Every user interface element that was unecessary was cut out. Buttons and banners were made bigger and easy to see. The sort by menu became a dropdown menu. The specific details of the movie became hidden in hover-states or button clicks.

Forums

While IMDb does allow a comments section, I thought a centralized forum would be a nice addition to the IMDb website as a whole. Again, the design is modern and completely simple as to make navigation a seamless and non-thinking process.
