Windi

About

Windi is a website / app which allows travelers to connect to people worldwide. Windi allows easier travel planning for groups and individuals who are looking to be a part of a travel group.

Problem

The problem in starting out this project was to decide whether there would be a need for a website / app like windi. Through preliminary research, I found out that there is a niche that has not yet been filled for a product which allows travelers specifically to connect and plan travels together.

User Persona

After a series of screener questions and interviews the user persona, "Karl" was created. Karl is a fictional persona that matches, in general, the would be user base for windi.

Competitive Analysis

A competitive analysis was done to figure out the strengths and weaknesses of each direct and indirect competitors as well as Windi itself.

Site Map

A simple and straightforward sitemap was made in order to organize the user touchpoints. The onboarding process as well as the site structure became more understandable through creating the site map.

User Narrative

A user narrative was created in order to visualize the onboarding process of our proto-persona "Karl". If I had more time, I would have done more user testing with real would-be users.

Solution

The problem of how best to onboard people was solved. Simplicity in the sign-up process through existing social-media platforms and a seamless transition to the app's home page was chosen as a design prerogative.

App

The windi app allows for easy travel planning through group creation, travel individual / group match, chat system, travel tracking, and travel gear sharing.
